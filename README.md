# heroku-flask
A minimal example for deploying Flask application on Heroku

## Steps

### Step 1

```cmd
git clone https://github.com/Henderake/heroku-flask.git
```

### Step 2

```cmd
heroku create [app-name]
```

### Step 3
Commit and push to heroku

```cmd
git commit
git push heroku master
```
